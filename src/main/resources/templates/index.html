To make the application work, use POSTMAN!

Endpoints:

/POST

Name: Json required format: Description:
/register { "username":"username", "password":"password" } Password need to be longer than 8 character.
/login { "username":"username", "password":"password" }
/item/add { "name": "testitem", "description":"testdescription", "photoUrl": "https://test.png", "startingPrice": 20, "purchasePrice": 40 } Photo url can be .png .jpg .bmp with http and https.
/add?balance={value} Add money by request param.
/item/{id} { "bid": 80 } Bid to item by id.

/GET

Name: Description:
/item/info/{id} Infos about specific item by id.
/item/list List first 20 sellable items.
/item/list/{pageNumber} List 20 sellable items by created date with page path variable.
